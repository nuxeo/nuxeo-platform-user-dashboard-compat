====================================
User Dashboard compatibility package
====================================

This module offers a compatibility with the old JSF dashboard on Nuxeo.

This feature has been reimplemented in Nuxeo to use content views, and a new
Opensocial dashboard is also available as of version 5.4.2.

This package is offered as a compatibility package for plugins that
used to customize this old dashboard, and has been removed as of Nuxeo
5.8 (see file DEPRECATED.txt)
